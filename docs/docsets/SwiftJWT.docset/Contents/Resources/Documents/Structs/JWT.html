JWT

public struct JWT

JSON Web Token with its header and claims.

Usage Example:

let jwt = JWT(header: Header([.typ:"JWT"]), claims: Claims([.name:"Kitura"]))
let signedJWT = jwt.sign(using: .rs256(key, .privateKey))

  • validateClaims(issuer:audience:)

    Validate the JWT claims. Various claims are validated if they are present in the Claims object. Various validations require an input. In these cases, if the claim in question exists and the input is provided the validation will be performed. Otherwise, the validation is skipped.

    The following claims are validated: iss, aud, azp, at_hash, exp, nbf, iat.
